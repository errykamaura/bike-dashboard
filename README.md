# 🚲 Bike Sharing Dashboard

Dashboard analisis data peminjaman sepeda menggunakan Streamlit berdasarkan dataset Bike Sharing (2011–2012).

---

## 📊 Project Overview
Project ini bertujuan untuk menganalisis pola peminjaman sepeda berdasarkan faktor:
- Cuaca
- Musim
- Hari kerja
- Tren waktu

Hasil analisis ditampilkan dalam bentuk dashboard interaktif menggunakan Streamlit.

---

## 🎯 Business Questions
1. Bagaimana pengaruh cuaca terhadap jumlah peminjaman sepeda?
2. Kapan waktu dengan jumlah peminjaman sepeda tertinggi?

---

## 📈 Key Insights
- Cuaca cerah meningkatkan jumlah peminjaman sepeda
- Musim Summer dan Fall memiliki demand tertinggi
- Hari kerja menunjukkan pola penggunaan sebagai transportasi utama (commuting)
